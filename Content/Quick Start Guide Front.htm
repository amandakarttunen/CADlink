Quick Start Guide

Before you get started...

This package includes:

  • USB key with the software license files
  • Installation DVD
  • Printers DVD (if you purchased the Print & Cut edition)

Note: If you purchase an upgrade package, use your existing USB key. You will not receive a new one.

Caution: If you import from design software and plan to install send-to plug-ins, install the other software prior to installing your CADlink product(s).

License Files

CADlink's product license files are on your USB key, which is your proof of purchase. If your USB key is lost or stolen, a new software package is required. This USB key is invisible to other applications.

In case of damage, contact CADlink about replacement options. If your product is still under warranty, fees are waived.

This is a step-by-step installation guide for installing your product. To learn more about your product, open the Getting Started topic of the help files after installation.

Installing SignLab EngraveLab ProfileLab

  1. Plug in your USB key with license files.
  2. Insert your installation DVD and follow the on-screen instructions.
  3. When prompted, select options. You will be guided through installing output devices, fonts, clip art, and any available patches.

    Note: Output devices and fonts can be installed later in the application. Go to File > Install. For instructions, use the Getting Started topic of the help file. If you are offline, insert your installation DVD.

  4. (If you purchased the Print & Cut edition) To install a printer, open SignLab and go to File > Launch Visual Production Manager and follow the on-screen instructions.

    Note: Printing devices can be installed later. Go to File > Launch Visual Production Manager. In the Visual Production Manager window, go to Devices > Manage Devices. If you are offline, insert your Printers DVD.

  5. Go to Help > Web Help > Getting Started to perform an output test.
